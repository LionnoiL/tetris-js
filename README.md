# Tetris JS

Простий клон класичної гри Tetris на чистому HTML, CSS і JavaScript. Проєкт створено як навчальний приклад роботи з DOM, подіями, анімацією та логікою гри.

## Що вже реалізовано

- ігрове поле 10 x 20;
- падіння тетроміно, переміщення вліво/вправо, прискорення вниз;
- поворот фігур із захистом від виходу за межі поля;
- видалення повних рядків із підрахунком очок;
- зростання рівня та швидкості гри;
- пауза/продовження гри;
- завершення гри, коли нова фігура не може з'явитися на полі.

## Керування

- Arrow Left — перемістити фігуру вліво;
- Arrow Right — перемістити фігуру вправо;
- Arrow Down — прискорити падіння;
- Arrow Up — повернути фігуру;
- Play / Pause — старт або пауза гри.

## Структура проєкту

```text
tetris/
├── index.html          # Головна сторінка гри
├── css/
│   └── style.css       # Стилі гри
├── js/
│   ├── main.js         # Основна логіка гри
│   ├── tetrominoes.js  # Опис фігур (тетроміно)
│   └── utils.js        # Допоміжні утиліти
└── images/
    └── sprites.svg     # Графічні ресурси
```

## Як запустити

Відкрийте файл [index.html](index.html) у браузері або запустіть локальний сервер, наприклад:

```bash
python3 -m http.server 8000
```

Потім перейдіть за адресою http://127.0.0.1:8000/.

## Відомі баги

- Під час гри є проблеми з поворотом деяких фігур, коли вони знаходяться біля краю ігрового поля.
- На завершальному етапі гри є неточності в поведінці гри, зокрема у момент завершення партії.
- Ці проблеми наразі залишаються без виправлення, оскільки основний фокус — документування поточного стану проєкту.

## Мета проєкту

Проєкт створюється для практики роботи з JavaScript, DOM, логікою гри та базовими анімаціями.
