# Tetris JS

Простий клон гри Tetris на чистому HTML, CSS і JavaScript. Проєкт створено як навчальний приклад роботи з DOM, подіями, анімацією та логікою гри.

## Що реалізовано

- ігрове поле 10 × 20;
- падіння тетроміно, переміщення вліво/вправо та прискорення вниз;
- поворот фігур із корекцією біля країв поля;
- видалення повних рядків із підрахунком очок;
- зростання рівня та швидкості гри;
- попередній перегляд наступної фігури;
- пауза/продовження гри;
- завершення гри, коли нова фігура не може з'явитися на полі;
- простий звук при падінні фігури.

## Керування

- Arrow Left — перемістити фігуру вліво;
- Arrow Right — перемістити фігуру вправо;
- Arrow Down — прискорити падіння;
- Arrow Up — повернути фігуру;
- Play / Pause — старт або пауза гри.

## Структура проєкту

```text
tetris/
├── index.html          # головна сторінка гри
├── css/
│   └── style.css       # стилі гри
├── js/
│   ├── config.js       # налаштування гри та параметри
│   ├── main.js         # основна логіка гри
│   ├── tetrominoes.js  # опис фігур (тетроміно)
│   └── utils.js        # допоміжні утиліти
├── images/
│   └── sprites.svg     # графічні ресурси
└── sounds/
    └── drop.wav        # звук падіння фігури
```

## Як запустити

Відкрийте файл [index.html](index.html) у браузері або запустіть локальний сервер, наприклад:

```bash
python3 -m http.server 8000
```

Після цього перейдіть за адресою http://127.0.0.1:8000/.
