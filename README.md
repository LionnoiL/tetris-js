# Тетріс

Навчальний проєкт для закріплення навичок роботи з JavaScript. Реалізація класичної гри "Тетріс" на чистому HTML, CSS та JavaScript, без використання сторонніх бібліотек чи фреймворків.

## Статус проєкту

Проєкт щойно розпочато (початковий коміт). Наразі закладено базову структуру файлів, функціонал буде додаватися поступово.

## Технології

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Структура проєкту

```
tetris/
├── index.html          # Головна сторінка гри
├── css/
│   └── style.css       # Стилі гри
├── js/
│   ├── main.js          # Основна логіка гри
│   └── tetrominoes.js   # Опис фігур (тетроміно)
└── images/
    └── sprites.svg       # Графічні ресурси
```

## Мета проєкту

Цей проєкт створюється з навчальною метою — для практики роботи з DOM, подіями, анімацією та логікою гри на JavaScript.

## Автор

Навчальний проєкт, розроблений самостійно в процесі вивчення JavaScript.
